Mô hình:
	- HMM bậc 1
	- mỗi từ loại được mô hình bằng 1 mô hình Markov ẩn 1 trạng thái
	- Xác suất chuyển trạng thái được tính dựa trên mô hình ngôn ngữ bigram
	
Chương trình chính làm những công việc sau:
	- Huấn luyện 2 bộ tham số của mô hình Markov ẩn: transimition matrix, emission matrix
	- Lưu ra file (model): T transition probability
						   E emission probability
	- Đánh giá: 
		đưa 1 câu đã gán nhãn vào, function evaluate sẽ tách lấy phần text và đưa vào gán nhãn bằng hàng tag(String)
		kết quả sau khi tag được so sánh với
